How To Use

The Katana Order Generator reads a customer Purchase Order — from a pasted email or an uploaded PDF — and creates a draft sales order in Katana for your review before submitting.

1 · Setup

1
Install the extension
Load the extension folder in Chrome via chrome://extensions → Load unpacked. Enable Developer mode first if it is not already on.
2
Add your Katana API key
Click the extension icon, then click Settings ↗ (or right-click the icon → Options). Paste your Katana API key and save. The extension reads your live product catalog — customers, variants, and locations — on every open.
Where to find your API key In Katana: Settings → Integrations → API → Generate API key.
3
Open the side panel
Click the extension icon in the Chrome toolbar. The side panel opens alongside your browser window. You'll see a status dot in the header — green means the catalog loaded successfully.

2 · Reading a Purchase Order

A
Paste Text tab — emails and plain-text POs
Copy the body of the customer's email or PO and paste it into the text area. Click Read Order. The parser will scan for customer name, SKUs, product names, quantities, PO reference number, and delivery date.
B
Upload PDF tab — PDF purchase orders
Switch to the Upload PDF tab. Drag your PDF onto the drop zone or click to browse. Text is extracted automatically from all pages using Y-coordinate grouping, so each table row is parsed in isolation — quantities and prices are pulled from the correct row regardless of how the PDF was encoded.
Scanned PDFs The extension reads text embedded in the PDF. Image-only (scanned) PDFs will not extract any text. The customer must provide a digital/native PDF, not a scan.

3 · Review & Confirm

After parsing, a review screen shows the extracted data. Check every field before creating the order — the parser is smart but not perfect.

1
Check the Customer field
The parser tries to match the customer name in the text against your Katana customer list. If it picks the wrong one (or none), use the dropdown to select the correct customer manually.
2
Verify Order No., PO Ref, and Delivery Date
The Order No. is auto-generated in SO-YYYYMMDD-XXXX format — you can edit it. The PO reference and delivery date are extracted from the document if present; fill them in manually if not.
3
Review the line items and confidence badges
Each matched item shows a confidence badge indicating how it was found:
Badge What it means Action needed
SKU Matched by exact SKU code Check qty & price — usually correct
Name Matched by product name Confirm it's the right product
Words Matched by keyword overlap — low confidence Verify carefully — may be wrong
Manual Added by you via search No action needed
Edit quantities and prices inline Click any qty or price field in the table to edit it. Use the × button to remove an item that was incorrectly matched.
4
Add missing items manually
Use the search box below the table to find and add any product the parser missed. Search by SKU or product name — select from the dropdown to add it with qty 1.
5
Create the order
Click Create Order. The extension sends the order directly to Katana via the API. A success screen confirms the order number. Click Create Another to start fresh.

4 · Tips & Known Limitations

  • PDFs work better than pasted text when the PO has a proper table layout — the PDF parser preserves row structure precisely.
  • SKU matching is case-sensitive. If a customer writes a SKU in lowercase and your catalog has it in uppercase, it won't match by SKU — but name matching will still attempt it.
  • Very short or generic product names (e.g. a product literally named "Box" or "Cable") can produce false word-level matches. Review Words badges carefully.
  • Scanned / image PDFs contain no embedded text and cannot be parsed. Ask the customer for a native digital PDF.
  • The catalog is fetched fresh each time the side panel opens, so new customers or products added to Katana are available immediately — no need to reload the extension.
  • Nothing is submitted to Katana until you click Create Order. You can freely review, edit, and reset without any risk of creating a draft order.
